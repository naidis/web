# Logseq vs Obsidian: Why Obsidian Users Add Naidis

**Logseq** and **Obsidian** are both excellent local-first note-taking apps. This isn't about trashing Logseq—it's about helping you understand what Naidis adds if you choose Obsidian.

## Logseq vs Obsidian: The Basics

Both tools share core values:

| Feature | Logseq | Obsidian |
|---------|--------|----------|
| **Local-first** | ✅ | ✅ |
| **Open Source** | ✅ (AGPL) | ⚠️ (closed, free for personal) |
| **Markdown Files** | ✅ | ✅ |
| **Bidirectional Links** | ✅ | ✅ |
| **Graph View** | ✅ | ✅ |
| **Offline** | ✅ | ✅ |
| **Plugins** | ✅ | ✅ (larger ecosystem) |
| **Price** | Free | Free (personal) |

## Why Choose Obsidian?

Users often pick Obsidian for:

- **Larger plugin ecosystem**: 1000+ community plugins
- **More flexible structure**: Documents, not just outlines
- **Themes and customization**: Extensive appearance options
- **Canvas**: Visual note mapping
- **Mobile apps**: Smoother iOS/Android experience
- **Active development**: Regular updates

## What Naidis Adds to Obsidian

If you're using or considering Obsidian, Naidis fills gaps that Logseq handles natively—plus adds unique capabilities.

### YouTube Transcripts

Extract full transcripts with timestamps. Create notes from videos automatically.

### Web Clipping

### RSS Reader

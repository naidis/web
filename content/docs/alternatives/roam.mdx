## What Naidis + Obsidian Does Better

### 1. Local Markdown Files

Your notes are plain markdown files on your computer. No proprietary database. Open them in any text editor, sync with any service, backup anywhere.

### 2. Cost Efficiency

- Roam: $165/year ongoing
- Obsidian + Naidis: $3.99/mo (Obsidian is free for personal use)

After 3 months, Naidis pays for itself.

### 3. AI-Powered Search

Ask questions about your notes in natural language. Naidis uses RAG to find relevant context and generate accurate answers.

### 4. Content Capture

Naidis brings content into your vault:
- YouTube transcripts with timestamps
- Web articles in clean markdown
- PDF text and tables
- Kindle highlights

### 5. Spaced Repetition

Built-in review system using SM-2 or Half-life algorithms. Better than Roam's Delta add-on, and included in the base price.

## What Roam Does Better

Roam still excels at:

- **Block-level references**: More mature implementation
- **Keyboard-driven workflow**: Power user shortcuts
- **Academic community**: Strong research user base
- **Queries**: Advanced block queries
- **Simplicity**: No plugins needed for core features

## Who Should Switch to Obsidian + Naidis?

Make the switch if:

- ✅ You want local data ownership
- ✅ You're tired of the subscription cost
- ✅ You want AI features
- ✅ You capture content from various sources
- ✅ You want extensibility via plugins
- ✅ You prefer markdown portability

## Who Should Stay with Roam?

Stay with Roam if:

- Block references are critical to your workflow
- You've built extensive queries in Roam
- You prefer a simpler, all-in-one tool
- Migration effort outweighs the benefits
- You're embedded in the Roam community

## Migration Guide

Moving from Roam to Obsidian:

1. Export your Roam database as Markdown
2. Use a Roam-to-Obsidian converter (several exist)
3. Move files to your Obsidian vault
4. Install Naidis for AI + content capture
5. Your bidirectional links will work automatically

# Welcome to Naidis

**Naidis** is the ultimate plugin for Obsidian. A unified Command Palette where you can perform all PKM tasks in a single interface.

## Key Features

- **Web Clipping** - Save web pages to Markdown
- **YouTube Transcript** - Extract and summarize YouTube transcripts
- **AI Chat** - AI chat using local LLMs
- **RSS Reader** - Subscribe to and read RSS feeds
- **PDF Processing** - PDF parsing, OCR, and table extraction

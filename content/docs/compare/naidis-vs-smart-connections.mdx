# Naidis vs Smart Connections

Both Naidis and Smart Connections bring AI to Obsidian. But they solve different problems. Here's how they compare.

## Quick Verdict

| If you want... | Choose |
|----------------|--------|
| **Discover note connections** | Smart Connections |
| **Free AI for Obsidian** | Smart Connections |
| **Content capture** | Naidis |
| **Spaced repetition** | Naidis |
| **YouTube/PDF/Web/Kindle** | Naidis |

## Different Goals

| Aspect | Smart Connections | Naidis |
|--------|-------------------|--------|
| **Primary Goal** | Find related notes | Capture & organize content |
| **Core Feature** | AI similarity matching | PKM toolkit |
| **AI Use** | Surface connections | Chat + search |

**Smart Connections**: "What notes are related to this one?"
**Naidis**: "Let me capture, organize, and review content."

## Use Case Recommendations

### Choose Smart Connections if:

1. **Discovery is your need**: You want to find connections between existing notes
2. **Free is important**: You don't want to pay for plugins
3. **You have many notes**: Smart View shines with large vaults
4. **Serendipity matters**: You enjoy unexpected note connections
5. **Simple setup**: Just want AI without extras

### Choose Naidis if:

1. **Content capture is primary**: YouTube, web, PDF, Kindle sources
2. **You want spaced repetition**: Review and retain highlights
3. **Comprehensive toolkit**: One plugin for many PKM needs
4. **You're okay paying**: $29 one-time is acceptable
5. **Structured learning**: You want to build mastery over time

## Using Both Together

These plugins complement each other:

- **Smart Connections**: Surfaces relationships in your vault
- **Naidis**: Captures content and helps you review it

Many users run both. There's minimal overlap.

Workflow example:
1. Use Naidis to capture YouTube transcript
2. Use Smart Connections to find related notes
3. Use Naidis spaced repetition to review key insights

## Our Take

**Smart Connections** does one thing exceptionally well: help you discover connections between notes. And it's free.

**Naidis** is a broader toolkit. Content capture, organization, review—with AI as one component.

They're not competitors. They're complements.
